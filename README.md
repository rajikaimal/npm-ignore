# npm-ignore

Generated .npmignore file

## Install

```
$ npm install @rajikaimal/npm-ignore
```

## Usage

```js
const npmIgnore = require("@rajikaimal/npm-ignore");

npmIgnore(".tmp").then(created => {
  console.log(created);
  t.true(created);
  t.end();
});
```
